* Mijn veronderstellingen:

* 
* Mijn conclusies:
< Iedere ouder is een vrouw of een man.

* Mijn veronderstellingen:
< Een moeder is waarschijnlijk een persoon.
< 
* 
* Mijn veronderstellingen:
< Een zoon is waarschijnlijk een persoon.
< Een kind is waarschijnlijk een persoon.
< 
* 
* Mijn conclusies:
< Ieder kind is een meisje, een vrouw, een jongen of een man.
* 
* Mijn veronderstellingen:
< Een dochter is waarschijnlijk een persoon.
< 
* 
* Mijn conclusies:
< Jan heeft een zoon, genaamd Johan.
< Anna heeft een zoon, genaamd Johan.
* 
* Mijn veronderstellingen:
< Johan is waarschijnlijk een persoon.
< Johan is waarschijnlijk een kind van Jan en Anna.
< Johan is waarschijnlijk deel van een gezin.
< Johan is waarschijnlijk geen meisje en geen vrouw.
< Johan is waarschijnlijk geen dochter.
< Johan heeft waarschijnlijk 2 ouders, genaamd Jan en Anna.
< Johan heeft mogelijk een moeder, genaamd Anna.
< Jan heeft waarschijnlijk een kind, genaamd Johan.
< Jan is waarschijnlijk een ouder van Johan.
< Jan is waarschijnlijk deel van een gezin.
< Jan is mogelijk een persoon.
< Anna heeft waarschijnlijk een kind, genaamd Johan.
< Anna is waarschijnlijk een ouder van Johan.
< Anna is waarschijnlijk deel van een gezin.
< Anna is mogelijk een persoon.
< Anna is mogelijk een moeder van Johan.
< Anna is mogelijk een vrouw.
< Anna is misschien geen man.
< Anna is misschien geen vader.
* 
* Mijn vragen:
< Is Johan een jongen of een man?
< Is Jan een man of een vrouw?
< Is Jan een vader of een moeder?
<
