# Organisms
每个生物都是动物、菌、人、微生物或厂。
# Animals
每个动物都是脊椎动物或无脊椎动物。
# Vertebrates
每个脊椎动物都是两栖动物、鸟、鱼、哺乳动物或爬虫。
# Invertebrates
每个无脊椎动物都是环节动物、节肢动物、棘皮动物、软体动物或原虫。
# Amphibians
无足目、青蛙、蝾螈、蝾和蟾蜍都是两栖动物。
# Birds
#Xxxx是鸟。
# Fish
#Xxxx是鱼。
# Reptiles
每个鳄鱼都是爬虫。
# Annelids
蚯蚓、蛔虫和扁虫都是环节动物。
# Arthropods
螃蟹、蜘蛛和昆虫都是节肢动物。
# Echinoderms
海星、海胆、沙美元和海参都是棘皮动物。
# Mollusks
蛤蜊，墨鱼，蛤、蚌、鹦鹉螺、章鱼、牡蛎、蜗牛和鱿鱼都是软体动物。
# Protozoa
变形虫和鞭毛虫都是原虫。
# Mammals
每个哺乳动物都是鲸类、有袋动物、啮齿动物、猴或人。
# Cetaceans
鲸鱼和海豚都是鲸类。
# Marsupials
袋鼠、考拉、负鼠和塔斯马尼亚恶魔都是有袋动物。
# Monkeys
倭黑猩猩、黑猩猩、大猩猩、狨猴、猩猩和松鼠猴都是猴。
# Rodents
海狸、豚鼠、仓鼠、老鼠、豪猪、鼠和松鼠都是啮齿动物。
# Arthropods
每个节肢动物都是蛛形纲、甲壳动物或昆虫。
# Arachnids
螨、蜱、蝎和蜘蛛都是蛛形纲。
# Crustaceans
螃蟹、龙虾和藤壶都是甲壳动物。
# Insects
甲虫、飞、蝴蝶、蛾、蜻蜓、蜜蜂、黄蜂和螳螂都是昆虫。
# Human-beings
每个人都是脊椎动物。
